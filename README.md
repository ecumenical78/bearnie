## 📚 网站核心内容速览

在这里，您可以快速找到那些直击痛点的实战教程：

- 🚀 **极速获取客户端**
  为您提供国内网络环境下免翻墙极速下载 GitHub 官方核心包（v2rayn-Core.zip）的多种技巧。
  
- 🔧 **节点测速与延迟分析**
  解析自带的“真连接延迟（Httping/Tcping）”与普通 Ping 的本质区别，教你筛选出带宽高、延迟低的极品服务器。
  
- ⚙️ **内核抉择：Xray 还是 Sing-box**
  从底层解剖客户端内核调用机制。对于 VLESS、Reality、Hysteria2 等新星协议，帮您匹配发挥出最佳性能的对应 Core 文件。
  
- 💻 **进阶玩法：开启 TUN 模式 (虚拟网卡)**
  从传统的“系统全局代理”升级为能拦截 Steam、Epic、Docker 和全局终端命令行的“真正的全局防泄漏网络代理”。
  
- 🛠️ **终极故障排查**
  代理突然变红？网页打不开？一直报错 `-1` 或者 `Timeout`？保姆式的排雷方案让您不再束手无策。

## 🛠️ 关于本项目源码

本代码仓库包含该教程站点的完整代码（基于 Astro + Tailwindcss 构建）。如果您希望协助我们编辑文章、改进功能或在本地进行编译预览：

### 本地部署运行

```bash
# 1. 把仓库克隆到本地
git clone https://github.com/YourUsername/bearnie.git

# 2. 安装全部依赖
npm install

# 3. 启动开发服务器
npm run dev
```

成功运行后，打开浏览器访问 `http://localhost:4321` 即可体验。
