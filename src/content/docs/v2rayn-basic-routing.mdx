## 什么是“路由 (Routing)”？

现在你的 V2RayN 已经成功连上了国外的服务器节点。但是，如果你此时直接上淘宝、看 B 站或者打国服网络游戏，你会发现页面加载极慢，游戏延迟突破天际。

为什么？因为默认情况下，所有流量不仅要绕道美国/香港，然后再绕回中国大陆，这白白浪费了机场流量，并且绕远路导致了极高的延迟卡顿。

这时候，我们需要一个“国内站直连，国外站走代理”的交通指挥员——这就是 **“路由分流策略”**。

---

## 开启“绕过大陆”分流基础模式

在 V2RayN 中，这一步非常简单，因为软件自带了完善的中国大陆 IP 和域名数据库：

1. 关注你的系统右下角，找到蓝色的 **V** 字图标（如果被折叠，请点开任务栏的 `^` 箭头）。
2. 用鼠标 **单击右键** 点击这个 V 图标，唤出菜单。
3. 找到 **“自动配置系统代理”** 并勾选它（此时图标变为纯蓝色）。
4. 找到 **“路由 (Routing)”** 菜单，将鼠标悬停在上面。
5. 在弹出的二级菜单中，选择 **“绕过大陆 (Bypass Mainland China)”**。

### 这四种路由模式分别代表什么？

当你悬停在路由菜单时，通常会看到以下几个主选项：

- **全局 (Global)**：
  所有的流量，不管你访问百度还是 Google，统统走代理节点出海。
  *适用场景：极其极端的网络审查时期，或者你想彻底伪装成纯海外用户的时候使用。绝不推荐日常挂载全局。*

- **绕过大陆 (Bypass Mainland China) 【最推荐日常使用】**：
  软件会拦截你的访问请求，如果发现你在请求 `baidu.com` 或者是国内的 IP，它就让这些流量“直连”过去（不走节点流量不消耗额度）。一旦发现你在请求被墙网站或是未被标记的境外 IP，就将其丢入加密代理通道。
  *适用场景：日常浏览、99% 的用户选这个准没错。*

- **黑名单模式/白名单模式**：
  需要你自己去 `参数设置 -> 路由设置` 里面手写非常复杂的规则。
  *适用场景：需要指定 ChatGPT 走美国节点、Netflix 走香港节点的高级折腾用户。*

## 如何检查分流是否生效？

当你设置完毕后，如果你不确定自己是否成功翻墙，可以进行一个小测试：

1. 打开浏览器，访问国内的测 IP 网站（如 [ip138.com](https://ip138.com) 或 [ip.cn](https://www.ip.cn)）。页面显示的应该是你 **真实的国内 IP 和城市名**。
2. 保持不要动 V2RayN 的设置。新建一个网页标签页，访问国外的测 IP 网站（如 [ipecho.net](https://ipecho.net) 或您直接用 Google 搜索 `what is my ip`）。如果显示的 IP 变成了你 **刚刚在 V2RayN 里选中的机场节点所在的国家（如日本/美国）**……

恭喜你！这说明“绕过大陆”分流完美生效，你既可以飞速看国内的优酷爱奇艺，同时可以随时在 Google 查资料而无缝切换了！
