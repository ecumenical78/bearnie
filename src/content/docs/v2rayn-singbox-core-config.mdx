## 前言

随着网络封锁技术的升级，传统的代理协议在面临高丢包、晚高峰骨干网拥堵时往往力不从心。**Hysteria 2（简称 Hy2）** 作为基于 UDP/QUIC 协议的下一代黑科技，凭借其强大的拥堵控制算法（BBR）和对丢包的高并发容忍度，成为了 2026 年极速冲浪的首选。

V2RayN 客户端原生支持通过挂载 **Sing-box** 内核来运行 Hysteria 2 节点。本文将手把手教你如何在 V2RayN 中升级内核并完成配置。

---

## 1. 检查并更新 V2RayN 的 Sing-box 内核

要正常运行 Hysteria 2 节点，V2RayN 目录中必须包含最新的 `sing-box.exe` 核心程序。

### 操作步骤：
1. 打开 V2RayN 客户端，点击顶部菜单栏的 **【检查更新】**。
2. 在下拉菜单中选择 **【检查更新 core】**（检查更新内核）。
3. 在弹出的内核列表中，找到 **`sing-box`**。
4. 如果有新版本，点击右侧的 **【下载/更新】** 按钮。V2RayN 会自动下载最新版本的 Sing-box 核心并将其解压替换至 `v2rayN/bin/sing_box/` 目录中。

> **提示**：如果由于网络原因自动更新失败，你可以直接访问 [Sing-box 官方 GitHub Releases](https://github.com/SagerNet/sing-box/releases) 页面，下载适用于 Windows (x86_64) 的 `sing-box-xxx-windows-amd64.zip` 包，解压后将其中的 `sing-box.exe` 手动覆盖放入 V2RayN 的 `bin/sing_box` 文件夹中。

---

## 2. 导入与添加 Hysteria 2 节点

你既可以通过导入机场提供的订阅链接自动生成 Hysteria 2 节点，也可以在 V2RayN 中手动创建。

### 方法一：通过订阅自动导入（推荐）
1. 复制机场提供的原生订阅链接（支持 V2Ray 格式或 Sing-box 格式）。
2. 在 V2RayN 中点击 **【订阅分组】** ➡️ **【订阅分组设置】** ➡️ 添加并保存链接。
3. 点击 **【订阅分组】** ➡️ **【更新订阅】**，节点列表中将自动拉取协议标识为 `Hysteria2` 或 `hy2` 的节点。

### 方法二：手动添加 Hysteria 2 节点
1. 在 V2RayN 主界面空白处右键，选择 **【服务器】** ➡️ **【添加自定义配置服务器】**。
2. 或者在顶部菜单点击 **【服务器】** ➡️ **【添加【Hysteria2】服务器】**（视 V2RayN 版本而定）。
3. 填入参数：
   - **地址 (Address)**：填入你的节点服务器域名或 IP。
   - **端口 (Port)**：填入主监听端口（例如 `443` 或端口跳跃范围如 `20000-40000`）。
   - **认证验证 (Password)**：填入你的认证密钥。
   - **传输层安全 (TLS)**：勾选启用。设置 `SNI`（伪装域名）与 `Insecure`（允许不安全证书，生产环境建议关闭）。

## 3. 高级进阶：端口跳跃 (Port Hopping) 设置

某些运营商（如中国移动或部分地区宽带）可能会对特定 UDP 端口（如 443）的高并发 Hysteria 2 流量进行阻断或 QoS 限速。开启端口跳跃可以有效避开限制：

1. 在编辑 Hysteria 2 节点时，将端口填入范围格式：`443,10000:65535`。
2. 此时，Sing-box 内核会在发送 UDP 数据包时自动在指定的端口段中频繁跳跃，使运营商防火墙无法对其进行特征分析和限速。

---

## 4. 常见报错与排错

### 报错一：`sing-box.exe failed to start`
* **原因**：解压目录包含中文，或者 `sing_box` 文件夹中缺少 `sing-box.exe` 主程序。
* **解决**：确保 V2RayN 的整个解压目录全为英文字符，并在【检查更新 core】中重新下载 Sing-box 内核。

### 报错二：测速真延迟显示 `-1` 或超时
* **原因**：UDP 流量被本地 Windows 防火墙或第三方杀毒软件（如 360/火绒）拦截。
* **解决**：在 Windows 安全中心中为 `sing-box.exe` 添加入站与出站放行规则。
