有时你的手机、平板或者电视盒子等设备可能不方便安装代理软件，但它们又处于和你的电脑同一个局域网（比如同一个 Wi-Fi）下。这时，你可以让运行了 V2RayN 的电脑充当一个**局域网代理网关**，让其他设备也能轻松科学上网。

## 开启局域网共享的步骤

### 第一步：在 V2RayN 中开启共享选项

1. 打开 V2RayN 软件界面。
2. 点击顶部菜单栏的 **“设置”** -> **“参数设置”**。
3. 在弹出的设置窗口中，找到 **“允许局域网的连接”**（Allow LAN connections）选项，并将其**勾选**。
4. 记下底部的端口号。默认情况下，Socks 端口是 `10808`，HTTP 端口是 `10809`。对于局域网共享，我们通常使用 HTTP 端口（`10809`）。
5. 点击 **“确认”** 保存设置。

### 第二步：获取本机的局域网 IP 地址

你需要知道这台运行 V2RayN 的电脑在当前 Wi-Fi/网络中的 IP 地址。
1. 在 Windows 系统中，按 `Win + R` 键，输入 `cmd` 并回车，打开命令提示符。
2. 输入 `ipconfig` 并回车。
3. 在输出的信息中，找到你当前使用的网络适配器（通常是“无线局域网适配器 WLAN”或“以太网适配器”）。
4. 找到 **“IPv4 地址”**，它的格式通常是 `192.168.x.x` 或 `10.x.x.x`（例如 `192.168.1.10`）。请将这个 IP 地址记下来。

### 第三步：在手机/其他设备上设置代理

以 iPhone 为例：
1. 确保手机和电脑连接在**同一个 Wi-Fi** 下。
2. 打开手机的 **设置** -> **无线局域网 (Wi-Fi)**。
3. 点击当前连接的 Wi-Fi 旁边的 **“i”** 图标，进入网络详情。
4. 滑到最底部，找到 **“配置代理”**，将其从“关闭”改为 **“手动”**。
5. 在**服务器**栏填入刚才记下的电脑 IP 地址（如 `192.168.1.10`）。
6. 在**端口**栏填入 V2RayN 的 HTTP 端口（默认为 `10809`）。
7. **认证**选项保持关闭即可。
8. 点击右上角的 **“存储”** 或 **“保存”**。

安卓设备或电视盒子的设置方法大同小异，都是在 Wi-Fi 的高级设置中找到“HTTP 代理”并填入对应 IP 和端口。

## 常见问题排查

如果你设置好后发现手机依然无法上网，请检查以下几点：

- **防火墙阻挡**：这是最常见的原因。Windows 防火墙可能会阻止局域网其他设备访问你的端口。你需要前往“Windows Defender 防火墙” -> “允许应用或功能通过 Windows Defender 防火墙”，找到 `v2rayN` 并勾选“专用”和“公用”网络；或者暂时关闭防火墙进行测试。
- **IP 地址变动**：如果路由器的 DHCP 会自动分配 IP，你电脑的局域网 IP 可能会发生变化。建议在路由器中将电脑的 IP 绑定为静态 IP，或者每次重新获取 IP 并更新手机端设置。
- **V2RayN 路由模式**：确保 V2RayN 的路由模式为“绕过大陆”或“全局”。

> **温馨提示**：局域网代理仅在你的电脑保持开机并运行 V2RayN 时有效。如果你关闭了电脑或带着电脑离开了当前网络，手机上的 Wi-Fi 代理设置会导致手机连不上网，届时记得将手机 Wi-Fi 里的“配置代理”改回“关闭”。
