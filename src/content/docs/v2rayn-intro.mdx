## 什么是 V2RayN？

V2RayN 是一个基于 V2Ray 内核的 Windows 桌面端图形化代理客户端。它支持多种主流协议（例如 VMess、VLESS、Trojan、Shadowsocks 等），并由于其开源、免费、稳定和功能强大，成为了众多用户的首选科学上网工具。

### 主要特点

- **支持多种协议**：兼容目前主流的大多数代理协议。
- **路由规则强大**：支持灵活的国内外分流，能够根据前缀、域名或 IP 自动决定走直连还是代理。
- **界面直观**：节点管理清晰，支持一键订阅更新。
- **开箱即用**：自带 V2Ray/Xray 内核，免去繁琐的命令行配置。

## 官方下载地址

请**务必**从官方 GitHub 仓库下载，以确保软件的安全性和纯净性。

- 官方 GitHub Release 页面：[https://github.com/2dust/v2rayN/releases](https://github.com/2dust/v2rayN/releases)

### 下载哪个版本？

在 Release 页面，你会看到多个压缩包：

1. **v2rayn-Core.zip**：包含 V2RayN 主程序以及常用内核（推荐新手下载这个版本，解压即用）。
2. **v2rayn.zip**：仅包含客户端界面，不包含内核。你需要自己手动去下载 V2Ray 或 Xray 内核并丢到对应文件夹中。

> **提示**：如果下载速度过慢，可以使用 GitHub 代加速或者临时开启现有代理进行下载。

## 安装与运行

1. 下载 `v2rayn-Core.zip` 压缩包。
2. 将压缩包解压到一个你容易找到的目录（例如 `D:\V2RayN`）。**注意：路径中最好不要包含中文字符，以免内核运行报错。**
3. 双击运行目录下的 `v2rayN.exe` 即可启动。
4. 首次启动时，如果你看到 Windows Defender 防火墙提示，请允许其通过。
5. 成功启动后，你能在任务栏右下角（系统托盘）看到一个蓝色的 **V** 字形小图标。
