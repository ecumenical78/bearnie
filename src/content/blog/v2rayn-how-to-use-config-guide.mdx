# V2RayN 怎么用？从零开始的 V2RayN 节点配置与上网指南

在完成了 [V2RayN 的下载与安装](/blog/v2rayn-download-install-tutorial) 后，您现在已经打开了软件的主页面。本文将指导您如何配置节点并成功翻墙。

## 1. 准备工作：拥有一个节点或订阅

V2RayN 只是一个工具/客户端，它本身不提供网络服务。要想科学上网，您必须有服务器节点信息。
节点信息的常见形式有：
- **单节点链接：** 例如以 `vmess://`、`vless://` 或 `trojan://` 开头的一长串字符。
- **订阅链接：** 机场服务商提供的一个 URL 网址，用于批量获取多个节点。

## 2. 导入节点信息

### 方式一：从剪贴板导入（适合单节点）
1. 复制别人分享给你的节点链接（例如 `vmess://...`）。
2. 在电脑右下角任务栏，找到 V2RayN 图标，**右键**点击。
3. 选择 **“从剪贴板导入批量URL”**。
4. 或者在主界面按下快捷键 `Ctrl + V` 完成导入。

### 方式二：导入订阅链接（最常用）
如果您购买了机场服务，通常会给您一个订阅链接。详细的导入步骤较长，请参考我们的专栏教程：
👉 **[《V2RayN 节点/订阅链接怎么导入？剪贴板与URL导入详解》](/blog/v2rayn-import-nodes-subscription)**

## 3. 测试与选择节点

节点导入后，会展示在 V2RayN 的列表中。
1. 点击选中一个节点，按下快捷键 `Enter` (回车)，将其设为**活动节点**（该行前面会出现一个蓝色的对勾）。
2. 选中节点并按快捷键 `Ctrl + R`，可以进行**真连接延迟测试**。
3. 列表最后“测试结果”那一列如果有数字（例如 `200ms`），说明这个节点是通的。如果是 `-1` 或红字提示，则可能不通。

## 4. 开启系统代理（核心步骤）

只选中节点是不够的，您还需要告诉 Windows 系统通过 V2RayN 发送网络请求。

1. 在电脑右下角任务栏，找到 V2RayN 小图标，**右键**点击。
2. 将光标移至 **“系统代理”** 选项。
3. 在弹出的二级菜单中，选择 **“自动配置系统代理”**。

此时，V2RayN 的系统托盘图标颜色会从【蓝色】变为【红色】（或粉色），这意味着代理已经成功开启！

## 5. 验证是否成功

打开浏览器，访问被屏蔽的网站（如 [Google](https://www.google.com) 或 [YouTube](https://www.youtube.com)）。如果能正常加载网页，恭喜您，配置成功！

## 6. 不用了记得关闭！

如果您不打算使用了，切记：
再次右键点击系统托盘的 V2RayN 图标，将 **“系统代理”** 改回 **“清除系统代理”**。托盘图标会变回蓝色。
如果不清除代理直接强制关机或关闭软件，可能会导致电脑无法正常上网（因为系统的网络仍在尝试走一个不存在的代理端口）。
