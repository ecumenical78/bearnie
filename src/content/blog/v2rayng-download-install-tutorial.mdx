## 📥 第一步：下载安装 v2rayNG

1.  在你的安卓手机上，打开浏览器访问官方 GitHub Release 页面：
    👉 **[https://github.com/2dust/v2rayNG/releases](https://github.com/2dust/v2rayNG/releases)**
2.  在最新版本（Latest 标签）下，点击展开 **Assets（资源列表）**。
3.  在文件列表中，找到并点击下载名为 **`v2rayNG_x.x.x_arm64-v8a.apk`** 的文件（推荐下载此版本，适配所有主流 64 位安卓机型）。
    > 如果你的手机是较老的 32 位机型（较为少见），则选择 `armeabi-v7a` 版本。
4.  下载完成后，打开安装包。系统可能会提示"允许安装来自未知来源的应用"，请在系统设置中为当前浏览器开启此权限，然后继续完成安装。

## 🔌 第二步：添加机场订阅

成功安装并打开 v2rayNG 后，你会看到一个几乎空白的主界面（节点列表为空）。接下来，我们通过添加机场订阅链接来导入节点：

1.  在主界面右上角，点击 **三个点（⋮）的菜单按钮**。
2.  在弹出的下拉菜单中，选择 **"订阅设置"（Subscription Setting）**。
3.  进入订阅管理页面后，点击右上角的 **"+"（添加）** 按钮。
4.  在弹出的对话框中：
    *   **备注（Remark）**：随意填写，例如"我的主力机场"。
    *   **地址（URL）**：将你从机场复制来的订阅链接完整粘贴到此处。
5.  点击右上角的 **保存（✓）** 图标。
6.  返回主界面，再次点击右上角的三点菜单，选择 **"更新订阅"（Update Subscription）**。
7.  等待几秒钟，节点列表将自动从机场服务器获取并填充所有可用节点。

---

## ⚡ 第三步：选择节点并开启连接

1.  订阅更新成功后，主界面将展示机场提供的所有节点列表。
2.  **（可选但推荐）测试延迟**：长按任意一个节点，在弹出的菜单中选择 **"测试全部配置（Real Delay）"**，等待测试完成，延迟数值将以绿/黄/红的颜色进行标注。
3.  **选择节点**：点击你想要使用的节点，节点左侧将出现选中标记。一般建议选择延迟（ms）显示较低且颜色为绿色的节点。
4.  **启动连接**：返回主界面，点击屏幕右下角的 **V 字形纸飞机图标（▷ 运行按钮）**，使其变为灰色停止按钮（三角形变为矩形）。
5.  首次开启时，安卓系统会弹出一个 **VPN 权限请求对话框**，提示"v2rayNG 请求建立 VPN 连接"，请点击 **"好"** 或 **"允许"** 继续。
6.  当主界面状态栏出现一把 🔑 钥匙图标时，代表连接已成功建立。打开浏览器访问 `google.com` 进行验证。

## ⚙️ 进阶配置：分流模式选择

在主界面左上角，你可以看到当前的 **"路由（Routing）"** 模式切换按钮。v2rayNG 提供以下几个常用的路由模式：

*   **绕过大陆与局域网（推荐）**：境外流量走代理，国内网站直连，兼顾速度与流量消耗的最佳平衡模式。
*   **全局（Global）**：所有流量，包括国内 App，都通过代理转发。会消耗更多节点流量，且国内 App 速度会略有下降。
*   **直连（Direct）**：关闭所有代理，等效于关闭科学上网。
