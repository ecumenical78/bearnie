# V2RayN 最新版官方下载地址与详细安装教程 (Windows篇)

V2RayN 是 Windows 平台上最受欢迎的免费开源科学上网客户端之一，支持 Vmess、Vless、Trojan、Shadowsocks 等多种代理协议。本文将为您提供详细的下载与安装指南。

## 1. 下载 V2RayN 客户端

请始终从官方渠道下载客户端，以确保软件安全无毒。

👉 **[官方 GitHub Release 页面下载地址](https://github.com/2dust/v2rayN/releases)**

**下载哪个版本？**
在发布的版本列表（Assets）中，您会看到多个压缩包：
- `v2rayN-Core.zip`：**（推荐新手下载）** 包含了 V2RayN 主程序以及常用的核心（如 Xray-core、v2fly-core）。下载解压后即可无需额外配置直接使用。
- `v2rayN.zip`：仅包含主程序，不包含核心。适合需要自己手动更新替换核心的高级用户。
- `v2rayN-With-Core-SelfContained.7z`：包含依赖环境（体积较大），如果您电脑没有或无法安装 .NET 环境，可以下载此版本。

## 2. 安装与运行

V2RayN 是免安装的绿色软件，下载后请按照以下步骤操作：

1. 将下载的 `v2rayN-Core.zip` 解压缩到一个**没有中文目录名**的文件夹中（例如 `D:\Software\v2rayN`）。
   > ⚠️ **注意：** 路径中带有中文可能会导致客户端或者内核运行报错！
2. 打开解压后的文件夹，双击运行名为 `v2rayN.exe` 的程序。
3. 如果系统弹出“Windows 已保护你的电脑”提示框（SmartScreen 拦截），请点击**“更多信息”**，然后选择**“仍要运行”**。

## 3. 常见报错：缺少 .NET 桌面运行时

运行 `v2rayN.exe` 时，如果弹窗提示“To run this application, you must install .NET Desktop Runtime”，说明您的系统缺少必要的运行环境。

**解决方法：**
1. 弹窗中通常会有一个“Yes”或“下载”按钮，点击它会自动跳转到微软官方下载页面。
2. 下载对应您系统的 x64 版本的 `.NET Desktop Runtime` 安装包。
3. 双击安装下好的 `.NET` 运行时，安装完成后，重新运行 `v2rayN.exe` 即可正常打开。

## 4. 界面认识与基本设置

成功打开 V2RayN 后，您会在屏幕右下角任务栏看到一个蓝色的“V”字形小图标（有时会被隐藏在托盘折叠菜单中 `^`）。

- **打开主界面**：双击任务栏的小图标。
- **退出软件**：右键点击小图标 -> 选择“退出”。
