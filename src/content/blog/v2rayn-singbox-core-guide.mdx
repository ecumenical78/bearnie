## 1. 为什么要使用 Sing-box 核心？

- **极致低能耗**：Sing-box 底层采用 Go 语言全新重构，内存占用仅为同类内核（如 Xray 或 Clash Meta）的 30% 到 50%，老旧机器运行也毫无压力。
- **协议兼容性强**：除了完美支持 Shadowsocks、Vmess、Vless 外，Sing-box 在处理新型 UDP 协议（如 Hysteria 2、TUIC v5）时表现出惊人的吞吐效率。
- **TUN 虚拟网卡完美搭档**：V2RayN 的虚拟网卡全局接管模式，底层首选驱动正是基于 Sing-box 实现的。

## 3. 在 V2RayN 中配置 Sing-box 核心

### 第一步：一键下载 / 更新 Sing-box 内核
1. 运行并打开 V2RayN 客户端主界面。
2. 在顶部菜单栏找到并点击 **【检查更新】**。
3. 在下拉菜单中选择 **【检查更新 SingboxCore】**。
4. 此时，V2RayN 会在底部日志框中输出下载进度，自动拉取最新的 `sing-box.exe` 文件并自动放置于软件的 `bin/sing-box` 文件夹中。
5. *提示：如果自动更新提示“下载失败”，请在右下角右击图标开启系统代理后再进行尝试，或者前往 Sing-box 的官方 GitHub 手动下载并放入对应的 `bin` 路径。*

### 第二步：指定节点协议核心
为了确保使用 Sing-box 解析节点：
1. 点击主菜单的 **【设置】** -> **【参数设置】**。
2. 切换至 **【核心基础设置】**。
3. 检查列表中所有的协议支持。如果您想让特定的协议（例如 Hysteria 2 或 VLESS）强制走 Sing-box 核心，可以在下拉框中将核心类型从 `Xray` 更改为 `sing-box`。
4. 点击 **【确定】** 保存并重启软件。

---

## 4. 开启 Sing-box 的终极绝招：TUN 模式

Sing-box 最强大的功能就是配合虚拟网卡实现无死角全局接管。

1. **以管理员身份运行 V2RayN**：退出 V2RayN，右键点击软件主程序图标，选择“以管理员身份运行”（这是创建虚拟网卡的硬性权限要求）。
2. **勾选开启 TUN**：在主界面底部，找到 **【开启 TUN】** 并勾选。
3. 此时底部的控制台会显示 Sing-box 服务创建 TUN 网络设备成功。电脑上的所有软件、包含各类终端命令行工具与外服游戏，都将无缝走代理。
