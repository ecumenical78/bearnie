# V2RayN 路由规则设置教程：如何实现全局代理与国内分流直连？

很多新手在使用 V2RayN 科学上网时会遇到一个痛点：开启代理后，访问外网（如 YouTube）很快，但回头聊微信、看 B站、使用国内 App 却变得非常卡顿，甚至提示 IP 异常。

这通常是因为没有配置好**路由（Routing）规则**。本文将教您如何设置路由，让 V2RayN 智能分流。

## 什么是路由规则（Routing）？

在 V2RayN 中，路由的作用就是扮演一个“交通警察”：
- 看到你去**国外网站**的流量，指挥它走**代理通道**（节点）。
- 看到你去**国内网站**或局域网的流量，指挥它走**直连通道**（你家的宽带）。

## 核心设置：选择路由模式

在 V2RayN 底部状态栏，或者右下角系统托盘右键菜单中，您都能找到 **【路由】(Routing)** 选项。点击它，通常有以下几个自带的预设模式：

### 1. 绕过局域网和大陆 (Bypass LAN and mainland China)
**（强烈推荐，平时的默认选择）**
- **作用**：国内网站（百度、淘宝、B站等）直接加载，不耗费机场流量资源；国外网站（谷歌、推特等）自动走节点代理。
- **优点**：兼顾速度与隐私，省流量，体验最佳。

### 2. 全局代理 (Global)
- **作用**：不管国内还是国外，电脑产生的所有网络流量全部强制发往你的远端服务器节点。
- **优点**：极少数特殊的网站或是国内也遭屏蔽的特殊场景下需要用到。
- **缺点**：浪费机场节点流量；访问国内网站延迟极高；很容易触发国内平台的异地登录风控（比如 QQ 提示异地冻结）。

### 3. 直连 (Direct)
- **作用**：所有流量强制不走代理，相当于关闭了科学上网功能。

## 进阶：如何自定义路由规则（添加黑/白名单）

有时预设的“绕过大陆”规则不够用。例如，某个冷门的外网没有被规则库收录，导致直连打不开；或者某个国内公司的内网域名被误判走了代理。您可以手动添加自定义规则。

1. 在 V2RayN 主界面，点击顶部菜单的 **【设置】** -> **【路由设置】(Routing setting)**。
2. 切换到 **【基础路由规则】** 或 **【高级路由规则】** 面板。
3. 您可以添加自己的规则集。例如，如果想让 `example.com` 强制走直连，可以在相关的 direct(直连) 规则下的“域名”输入框中，加入 `domain:example.com`。
4. 如果想让某个 IP 强制走代理，可以在 proxy(代理) 规则的“IP”框中，加入特定的 IP 段。

*(注：现代版本的 V2RayN 通常带有很完善的 `geoip.dat` 和 `geosite.dat` 规则库，普通用户通常只需选择 **“绕过局域网和大陆”** 即可，无需手动折腾自定义规则。)*

## 路由失效？检查“系统代理”模式

请务必注意：**路由规则只有在流量成功进入 V2RayN 之后才会生效。**
如果您发现选了“绕过大陆”但似乎还是全局代理一样卡，请检查您的系统代理设置。任务栏右键 V2RayN 图标：
- 确保 **“系统代理”** 选的是 **“自动配置系统代理”**(图标变红)。

完成上面的配置，您就可以无缝畅游互联网，享受国内外秒开的极致体验了！
