# V2RayN 路由分流规则与自定义黑白名单高级玩法 (2026实战篇)

很多人在使用 V2RayN 时，只会使用右下角托盘最基础的“绕过大陆”功能。但这在 2026 年越来越复杂的互联网环境下往往不够用。比如：您可能希望看 Netflix 时走香港节点，用 ChatGPT 时走美国节点，而玩国服游戏时完全直连。

这就是 **“路由分流 (Routing)”** 大显身手的地方。

## 什么是路由分流？

简单来说，路由分流就像是您电脑网络的一个“交警”。当您的浏览器或软件发出一个网络请求（比如访问 `google.com` 或 `bilibili.com`）时，V2RayN 会根据您设定好的规则判断：
- 这个请求应该被发送给代理节点（走梯子）。
- 这个请求应该直接发给本地运营商（直连）。
- 或者这个请求干脆被丢弃（拦截广告）。

## 基础设置：路由规则集简介

在 V2RayN 菜单栏点击 **“设置” -> “路由设置”**，您将看到“基础路由”和“高级路由”的选项。

V2RayN 默认内置了最经典的基础路由集合：
*   **全局 (Global)**：强制所有流量走代理。
*   **绕过大陆 (Bypass Mainland China)**：根据内置的中国大陆 IP 库和域名库（`geoip:cn` 和 `geosite:cn`），自动让国内请求直连，其余流量走代理。

如果这已经满足您的需求，您只需在系统托盘右击图标，将路由设置为“绕过大陆”即可。

## 高级实战：编写自定义分流规则

如果您需要更精细的控制，就需要用到**“高级路由”**。

点击 **“高级路由”** 选项卡，点击左上角的“添加”按钮。每一个新建的规则我们称为一个“路由集”。

### 实战一：让 ChatGPT 强制走特定节点

由于 OpenAI 经常封禁特定机房的 IP，您可能买了一个很贵但纯净的“美国原生节点”专门用来访问 ChatGPT，而平时看视频用便宜的“香港节点”。

1. 在系统主界面找到那个“原生节点”，记住它的**“别名”**或**“内部ID（tag）”**（假设内部 TAG 叫 `proxy-us`）。
2. 在路由设置的**白名单/代理**一栏中，添加匹配 OpenAI 的规则，例如输入：
   `domain:openai.com, domain:chatgpt.com`
3. 在出口选择中，您可以通过自定义 JSON 将符合上述域名的流量，单独绑定（outboundTag）到您刚才那个 `proxy-us` 节点。

> **小贴士**：大部分普通用户无需去学繁琐的 JSON 代码，现在的机场通常会提供已经写好分流模块的配置文件（如 V2Ray 的 `config.json` 或 Clash 分流订阅转换）。

### 实战二：屏蔽烦人的弹窗广告与追踪软件

利用路由的 `block`（拦截）机制，V2RayN 也可以当做一个强力的广告拦截器：

1. 添加一条新规则。
2. 目标地址填入广告商的域名，例如：`domain:adsystem.com, keyword:guanggao`
3. 将该规则的“出口 (Outbound)”设置为 `block`。
这样，当软件企图加载这些广告时，V2RayN 就会像“黑洞”一样直接吃掉这些请求，实现全局去广告。

## 2026 年新特性：应用进程级分流（Process Based Routing）

在最新的 Xray 内核加持下，V2RayN 现已成熟支持基于**可执行文件（exe）名称**的进程分流。

如果您想让某个游戏完全直连，不影响对战延迟：
仅仅在白名单规则里利用 `app` 标签即可：
```text
app:TslGame.exe
app:League of Legends.exe
```
将这类进程绑定的出口设为 `direct`（直连）。结合**TUN虚拟网卡模式**，它能完美实现“一半游戏直连本地，一半工具走海外代理”的电竞级苛刻要求！
