# V2RayN 节点/订阅链接怎么导入？剪贴板与URL导入详解

在配置好 V2RayN 环境后，获取并导入节点是实现科学上网的最关键一步。本文将详细介绍**单节点导入**和**订阅链接导入**两种常见方式。

---

## 方式一：导入单节点（剪贴板导入）

如果您从朋友那里获取了一个分享链接（通常以 `vmess://`, `vless://`, `ss://`, `trojan://`, 等开头），可以使用此方法。

### 操作步骤
1. **复制链接**：用鼠标选中那段很长的分享链接，右键选择“复制”（或使用 `Ctrl + C`）。
2. **打开 V2RayN**：双击任务栏右下角的 V2RayN 图标打开主界面。
3. **导入节点**：
   - 快捷键法：直接在软件主界面按 `Ctrl + V`。
   - 菜单法：点击顶部菜单栏的 **【服务器】** -> 选择 **【从剪贴板导入批量URL】**。
4. 此时，你的 V2RayN 列表里应该多出了一行服务器信息。选中它，按 `Enter` 键将其设为活动节点即可。

---

## 方式二：导入订阅链接（最常用推荐）

如果您购买了机场服务，服务商通常会提供一个“订阅链接”（一个以 `http` 或 `https` 开头的网址）。这种方式可以一次性导入几十个节点，并且支持随时更新更换失效节点。

### 第一步：添加订阅设置
1. 打开 V2RayN 客户端主界面。
2. 点击顶部菜单栏的 **【订阅】** -> **【订阅设置】**。
3. 在弹出的窗口中点击左下角的 **【添加】(Add)** 按钮。
4. 填写订阅信息：
   - **备注 (Remarks)**：随便填，比如“我的机场”或机场的名字。
   - **地址 (Url)**：将你从机场后台复制的“V2Ray订阅链接”粘贴到这里。
   - *(可选) Enable (启用)*：确保那一列打上了勾。
5. 点击右下角的 **【确定】(OK)** 保存设置。

### 第二步：更新订阅获取节点
添加完设置后，主界面并不会立刻出现节点。
1. 回到主界面，点击顶部菜单栏的 **【订阅】**。
2. 选择 **【更新订阅】(Update subscription)**。
3. 如果机场链接没问题且您的网络能连通，下方列表就会立刻刷出大量节点！

---

## 进阶技巧：开启“自动更新订阅”

为了防止节点失效，您可以让 V2RayN 每次启动时自动去机场拉取最新的节点列表。

**设置方法：**
1. 点击主界面的 **【设置】** -> **【参数设置】**。
2. 切换到 **【V2rayN设置】** 标签页。
3. 勾选选项 **【随软件启动自动更新订阅】(Update subscription on startup)**。
4. 点击 **【确认】** 保存。
